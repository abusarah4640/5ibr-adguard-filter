# Release Notes - v1.0.0-beta

Release type: Project Stabilization / Beta

## Summary

This beta marks the transition from a filter-building script into a structured open-source toolkit for managing a filter database, review workflow and reproducible releases.

## Included

- Command registry based CLI.
- Database management commands.
- Services layer.
- Central database integrity checks.
- Review workflow.
- Metadata-aware CSV handling.
- Specs and ADR documentation.

## Verification Checklist

```bash
pytest -q
python fivebr.py build
python fivebr.py validate
python fivebr.py doctor
```

## Known Limitations

- No REST API yet.
- No Web UI yet.
- No plugin system yet.
- CSV remains the source of truth.

These are intentional limitations for stabilization before expanding the platform.
